SCENE / 02
"就三个错别字和一个数字,别让我为这个装个编辑器。"

想顺手改几个字

[ 丑话说前头 ]

它做不到什么

先说清楚,省得你踩。

首次要联网

排版引擎（Mermaid、公式、代码高亮这些）是从 CDN 拉的,第一次打开需要网络,之后浏览器缓存了能离线用一阵。但你的文档内容始终不经过任何服务器——读、渲染、保存全在本机。

写回挑浏览器

静默写回原文件依赖 File System Access API,目前只有 Chrome / Edge 这类浏览器支持。Safari 和 Firefox 会退回"下载一份,你自己覆盖"。

改动记录认路径

"你不在时文件变了哪儿"靠文件路径来记。两个不同文件夹里的同名文件会串在一起——浏览器不给绝对路径,这个绕不过去。

超大文件只报"变过"

单篇超过 128KB（约 6 万字中文）时只存指纹,能告诉你它被改过,但定位不到具体改在哪儿——界面上会直说。

整段重写时 diff 会碎

词级对比是字符级算法,整段推倒重写时它会去匹配零星的"的""。",看起来有点碎。Word 也这样,是这类算法的固有行为。
